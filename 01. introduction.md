# Introduction to Red Teaming 

**Red Teaming** is a **full-scope adversary simulation** designed to test how well an organisation can **detect, respond to, and recover from real cyber attacks**. In simple terms, it means **acting like a real attacker** and seeing how far you can go **before someone notices**.

It is not about breaking everything.
It is about answering one simple question:

> *If a real attacker targeted this company, would anyone catch them in time?*
